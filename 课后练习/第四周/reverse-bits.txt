leetcode:190 颠倒二进制位
颠倒给定的 32 位无符号整数的二进制位。

解法一 对称位构造
复杂度分析：时间复杂度O(1)  空间复杂度O(1)
 
public class Solution {
    //对称位构造 如果某一位是 1 的话，则将答案相应的对称位置修改为 1。
    public int reverseBits(int n) {
        int res = 0;
        // 从低位 循环检查每个位置是否为1
        for (int i = 0; i < 32; i++){
            //为1则修改res对应对称位置为1
            if (((n >> i) & 1) == 1){
                res |= (1 << (31 -i));
            }
        }
        return res;
    }
}
